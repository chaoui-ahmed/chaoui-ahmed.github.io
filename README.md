My personal portfolio website showcasing my engineering projects, academic background at Eurecom, professional experiences, and core software engineering skills. 

Built with modern web development practices including Next.js App Router, TypeScript, Tailwind CSS, and fluid interactive animations powered by Framer Motion.

---

## ⚡ Features

- **Dynamic Interactive Resume**: Clean timeline of education (Eurecom engineering school, CPGE prep school) and professional tutoring/facilitating experience.
- **Multilingual Support**: Fully togglable localized content (English and French versions).
- **Responsive Layout**: Designed first for mobile screens, scaling elegantly to tablets and desktop displays.
- **Modern UI Components**: Sleek dark mode aesthetics featuring interactive cards, buttons, and custom layout structures.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Frontend**: [React 18](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [Radix UI](https://www.radix-ui.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js 18+ and `pnpm` installed.

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/chaoui-ahmed/chaoui-ahmed.github.io.git
   cd chaoui-ahmed.github.io
   ```

2. **Install dependencies**:
   ```bash
   pnpm install
   ```

3. **Run the development server**:
   ```bash
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.
